NUEVA FUNCIONALIDAD IMPLEMENTADA
=================================

El compresor ahora DETECTA AUTOMÁTICAMENTE el tipo de archivo 
original y lo GUARDA dentro del archivo .lz78.

¿Cómo funciona?
---------------

1. AL COMPRIMIR:
   - El programa lee la extensión del archivo (.txt, .docx, .jpg, etc.)
   - Guarda esta extensión en el archivo .lz78
   - Formato: [Header][Extensión][Diccionario][Datos]

2. AL DESCOMPRIMIR:
   - Lee la extensión original del archivo .lz78
   - Muestra el tipo de archivo detectado
   - Sugiere automáticamente el nombre correcto con la extensión

3. AL GUARDAR DESCOMPRIMIDO:
   - Nombre sugerido: "archivo_descomprimido.EXTENSIÓN_ORIGINAL"
   - Ejemplo: "documento.docx" → "documento_descomprimido.docx"
   - Filtro específico para ese tipo de archivo

PRUEBAS REALIZABLES
====================

Archivo de texto (.txt):
  - Comprime "prueba.txt"
  - Genera "prueba.lz78"
  - Descomprime y sugiere "prueba_descomprimido.txt"

Documento Word (.docx):
  - Comprime "reporte.docx"
  - Genera "reporte.lz78"
  - Descomprime y sugiere "reporte_descomprimido.docx"
  - El archivo restaurado funciona igual que el original

Imagen (.jpg):
  - Comprime "foto.jpg"
  - Genera "foto.lz78"
  - Descomprime y sugiere "foto_descomprimido.jpg"
  - La imagen se ve perfectamente

PDF (.pdf):
  - Comprime "manual.pdf"
  - Genera "manual.lz78"
  - Descomprime y sugiere "manual_descomprimido.pdf"
  - El PDF funciona correctamente

INFORMACIÓN MOSTRADA AL DESCOMPRIMIR
=====================================

Para archivos de TEXTO:
  - Diccionario generado
  - Contenido completo del archivo

Para archivos BINARIOS:
  - Tipo de archivo detectado
  - Extensión original
  - Tamaño del archivo
  - Diccionario generado
  - Mensaje indicando que es binario

COMPATIBILIDAD
==============

✓ Archivos comprimidos CON extensión guardada (nuevos)
✓ Archivos comprimidos SIN extensión guardada (antiguos)
  - Si no hay extensión guardada, usa .txt por defecto

VENTAJAS
========

1. No necesitas recordar qué tipo de archivo era
2. Automáticamente sugiere el nombre correcto
3. No hay riesgo de guardar con extensión incorrecta
4. Compatible con CUALQUIER tipo de archivo
5. Totalmente transparente para el usuario
