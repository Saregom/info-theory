README - Compresor LZ78
========================

Este es un compresor/descompresor basado en el algoritmo LZ78.

Características:
- Compresión de archivos de texto
- Compresión de archivos binarios (Word, Excel, PDF, imágenes)
- Visualización del diccionario generado
- Estadísticas de compresión

Tipos de archivos soportados:
- Texto: .txt, .log, .csv, .xml, .json, .html
- Documentos: .doc, .docx, .pdf
- Hojas de cálculo: .xls, .xlsx
- Imágenes: .jpg, .png, .gif, .bmp
- Y cualquier otro archivo binario

Uso:
1. Cargar archivo
2. Comprimir
3. Guardar comprimido (.lz78)
4. Cargar y descomprimir
5. Guardar descomprimido
